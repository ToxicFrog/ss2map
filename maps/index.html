Automatically generated, browser-viewable, searchable maps for:
  • System Shock
  • System Shock 2 (v2.48 with SCP beta 5 installed)
  • Thief Gold (v1.26 with TFix 1.27 installed)
  • Thief 2: The Metal Age (v1.26 with T2Fix 1.27e installed)
Also, some maps for fan-missions:
  • System Shock: Rewired
  • System Shock 2: Ponterbee Station
  • System Shock 2: UNN Nightwalker
  • System Shock 2: UNN Polaris

The SS1 and Rewired maps are generated using ss1edit's map tool.

The SS2 and Thief maps are generated using mismap.
You can also download any of the maps as zip files for offline viewing.
